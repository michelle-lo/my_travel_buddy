# My Travel Buddy

## Quick Installation

### Install virtual environment if you haven't yet
```
pip3 install virtualenv
```

### git clone the repo and cd to it or whatevs

### Create virtual env
```
python3 -m venv myenv
source myenv/bin/activate
```

### Install Flask
```
pip3 install flask
export FLASK_APP=hello
```

### Run
```
python3 main.py
```
